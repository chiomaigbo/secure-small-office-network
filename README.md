# Secure Small Office Network

## Overview

This project demonstrates the design and implementation of a secure small office network using industry best practices. It was completed as part of the TechCrush Cybersecurity Training Capstone.

---

## Objectives

- Design a secure office network
- Segment traffic using VLANs
- Configure firewall rules
- Perform vulnerability scanning
- Analyze network traffic
- Improve network security posture

---

## Technologies Used

- Cisco Packet Tracer
- Wireshark
- Nmap
- Windows Firewall
- TCP/IP
- VLANs

---

## Skills Demonstrated

- Network Security
- Vulnerability Assessment
- Firewall Configuration
- Network Analysis
- Security Documentation

## Results

Successfully designed and secured a simulated office network, validated firewall configurations, performed network reconnaissance with Nmap, and analyzed traffic using Wireshark to verify expected network behavior.
