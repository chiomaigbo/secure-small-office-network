# Secure Small Office Network

## Project Overview

This project was completed as part of the TechCrush Cybersecurity Capstone Project. The objective was to design, implement, and secure a simulated small office network using Cisco Packet Tracer and validate security controls through network analysis and testing.

## Objectives

- Design a secure office network topology
- Implement network segmentation
- Configure password-based access controls
- Apply firewall rules
- Perform port scanning using Nmap/Zenmap
- Analyze network traffic with Wireshark
- Demonstrate mitigation of common network threats

## Technologies Used

- Cisco Packet Tracer
- Nmap / Zenmap
- Wireshark
- Windows Defender Firewall
- TCP/IP
- VLAN Concepts
- Network Segmentation

## Results
The project successfully demonstrated how layered security controls—including segmentation, password protection, firewall configuration, and port hardening—can reduce the attack surface of a small office network while maintaining secure internal communication.
